Zoomで共有した音が相手に聞こえないときのチェックリスト

動画やBGMを画面共有しても「音が聞こえません」と言われたら、上から順に確認してください。だいたいSTEP 1〜3で直ります。

0 / 0 確認済み
基本まずここを確認（ほとんどはこれで直る）
  1. 画面共有を開始する緑のボタン 共有 を押す
  2. 共有する画面を選ぶウィンドウの左下にある 音声を共有 にチェック
  3. すでに共有中の場合は、画面上部のメニュー 詳細（…）→ 音声を共有 をオン
バージョンによって「サウンドを共有」「コンピューターの音声を共有」という表記の場合もあります。
  1. 再生する動画・YouTube・PowerPoint側の音量がミュート/0になっていないか
  2. タスクバー右下のスピーカーアイコンで、Windows全体の音量を確認
  3. スピーカーアイコンを右クリック → 音量ミキサーを開く で、アプリごとの音量が0になっていないか
自分のPCで音が鳴っていないものは、相手にも届きません。まず自分に聞こえる状態にしてから共有します。
  1. Zoomを開き、右上の自分のアイコンをクリック
  2. アップデートを確認 を選び、最新版（Zoom Workplace 7.x）に更新
  3. 更新後はZoomを再起動してから共有を試す
応用それでもダメなら（Windows特有の原因）

基本の3つで直らない場合、Zoomの設定・Windowsのオーディオ機能が原因のことがあります。

  1. ホスト（主催者）が、共有ボタン横の ^（上矢印）をクリック
  2. 同時に1名の参加者が共有可能 を選択
「音声を共有」のチェックボックスがグレーアウトして押せないときは、ほぼこれが原因です。
  1. 設定 → システム → サウンド → 使用中のスピーカーを選択
  2. オーディオの強化（オーディオ拡張機能）を オフ にする
  3. 「Realtek Audio Console」が入っているPCは、そちらのノイズ抑制・音響効果もオフにする
  1. デバイスマネージャー → オーディオの入力および出力 に「ZoomAudioDevice」があるか確認
  2. 無い・⚠マークが付いている場合は、Zoomをアンインストール
  3. zoom.us/download から最新版を再インストール（管理者権限で実行）
  1. いったんBluetooth機器を切断し、PC本体のスピーカーで共有を試す
  2. それで直る場合は、Zoomの 設定 → オーディオ でスピーカーとマイクの組み合わせを見直す
別解音だけ流したい・もっと良くしたい

🎵 画面はいらない、BGM・音声だけ流したいとき

  1. 共有 ボタン → 詳細 タブを開く
  2. コンピューターオーディオ（コンピューターの音声のみ）を選んで共有

画面を共有せずPCの音だけを流せます。マイクとは別系統なので、しゃべりながらBGMを流す用途にも使えます。仮想オーディオソフト（VB-Cable / VoiceMeeter）を入れなくても、いまはこれで足ります。

🎧 音質を上げたいとき（音楽・動画上映向け）

  1. Zoomの 設定 → オーディオ → 詳細 を開く
  2. ミーティング内オプションをマイクから「オリジナルサウンド」に切り替えられるように表示 をオン
  3. 高忠実度音楽モード と ステレオオーディオ にチェック

※高音質モードは通信量が増えるので、回線が不安定な場合はオフのままを推奨します。

最終手段STEP 7まで全部ダメだったとき

会社管理のPCで設定が制限されている、原因不明だが今日のミーティングは乗り切りたい——そんなときは、Zoomのサウンド共有機能を使わずに音を届けます。

🔧 VB-CABLEで強引に解決する（サウンド共有を完全バイパス）

無料の仮想オーディオドライバ「VB-CABLE」を使い、PCの音をマイク入力としてZoomに流し込む方法です。Zoomからは「マイクでしゃべっている」ようにしか見えないため、サウンド共有まわりの不具合が何であっても確実に音が届きます。

PCの音 → CABLE Input（仮想スピーカー）→ CABLE Output（仮想マイク）→ Zoom

手順1：VB-CABLEをインストール（初回のみ）

  1. vb-audio.com/Cable から「VB-CABLE Driver」をダウンロードしてZIPを展開
  2. VBCABLE_Setup_x64.exe を右クリック → 管理者として実行 → Install Driver
  3. PCを再起動（再起動しないとデバイスが正しく認識されません）

手順2：PCの音の出力先をCABLEに切り替える

  1. タスクバー右下のスピーカーアイコンをクリック
  2. 出力デバイスを CABLE Input (VB-Audio Virtual Cable) に変更
この時点で自分のPCから音が聞こえなくなりますが正常です（音はケーブルの中に流れています）。自分でも聞きたい場合は手順4へ。

手順3：ZoomのマイクをCABLEにする

  1. Zoomの 設定 → オーディオ を開く
  2. マイクを CABLE Output (VB-Audio Virtual Cable) に変更
  3. マイク音量を自動調整します のチェックを外す
  4. 背景雑音を抑制 を 低 にする（music・BGMがノイズ扱いで消されるのを防ぐ）
  5. 可能なら オリジナルサウンド をオンにして音質劣化を防ぐ

手順4：自分でも音を聞けるようにする（任意）

  1. スピーカーアイコンを右クリック → サウンドの設定 → サウンドの詳細設定（サウンドコントロールパネル）
  2. 録音 タブ → CABLE Output をダブルクリック
  3. 聴く タブ → このデバイスを聴く にチェック → 再生先に普段のスピーカー/イヤホンを選択 → OK

手順5：自分の声も一緒に届けたいとき（任意）

手順3でマイクをCABLEにしたため、そのままだと自分の声が相手に届きません。しゃべりながら音も流したい場合：

  1. サウンドコントロールパネルの 録音 タブ → 実際のマイクをダブルクリック
  2. 聴く タブ → このデバイスを聴く にチェック → 再生先を CABLE Input にする → OK

これで「PCの音＋自分の声」が両方ケーブルに混ざってZoomへ届きます。

終わったら必ず元に戻すこと：①Windowsの出力デバイスを普段のスピーカーに戻す ②Zoomのマイクを実際のマイクに戻す ③手順4・5で入れた「このデバイスを聴く」のチェックを外す。戻し忘れると、次の会議で「声が聞こえない」事故になります。
複数の音源を個別に音量調整したい・常用したい場合は、上位ツールの VoiceMeeter（同じ作者の無料ミキサー）を検討してください。この方法はあくまで「今日を乗り切る」ための応急処置です。
