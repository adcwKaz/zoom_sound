# zoom_sound

Zoomの画面共有で「音が相手に聞こえない」問題（Windows向け）を、上から順に試すだけで解決できるチェックリストページ。

- 公開URL: https://adcwkaz.github.io/zoom_sound/
- 単一の `index.html`（依存ライブラリなし）
- チェック状態はブラウザの localStorage に保存

## 内容

1. **基本** — 「音声を共有」チェック / 音源側のミュート / Zoomのアップデート
2. **応用** — 同時共有設定 / ノイズ除去の干渉 / ZoomAudioDeviceの修復 / Bluetooth
3. **別解** — コンピューターオーディオのみ共有 / 高忠実度音楽モード
4. **最終手段** — VB-CABLEでサウンド共有をバイパスし、マイク入力として音を流す手順
